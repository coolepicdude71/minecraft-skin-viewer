![Minecraft Skin Viewer Banner](BANNER.png)

A simple web app that lets you type a Minecraft username and instantly see the player's skin.

The viewer uses the Minecraft skin service from mc-heads.

---

## 🚀 Live Demo

Try the viewer here:

👉 https://coolepicdude71.github.io/minecraft-skin-viewer

---

## 🧩 How It Works

1. Enter a Minecraft username.
2. Click **Show Skin**.
3. The app loads the character skin using an online skin API.
4. The skin appears instantly on the page.

---

## 🛠 Run Locally

1. Clone the repository

```
git clone https://github.com/YOUR-USERNAME/minecraft-skin-viewer.git
```

2. Open the folder.

3. Open `index.html` in your browser.

That's it!
